Beschreibung des Tools (Schwepunkt, Zielgruppe)

Die Webanwendung für selbstorganisierte Eventplanung ermöglicht registrierten Benutzern Events zu erstellen und mit anderen Benutzern zu teilen, bzw. sie einzuladen.

Es erleichert die Organisation von Party's, Geburtstagen, Meetings und anderen Zusammentreffen mit Freunden oder Kollegen.

Das Tool ist für jedermann gedacht, es soll einfach zu bedienen sein, sodass man in kurzer Zeit ein Event erstellen und Freunde und Bekannte einladen kann.

Funktionsumfang

  • Startseite mit Login

    Benutzer ruft die Seite auf und wird informiert, was Comeet ist und kann sich als Nutzer einloggen oder einen neuen Account erstellen

  • Eingeloggte User können ihre Events sehen und die Events von Ihren Freunden

  • Eingeloggte User können rechts sehen welche Freunde sie haben

  • Neue User können sich registrieren

  • Eingeloggte User können ein Neues Event anlegen

    • Wann - Datum/Uhrzeit (Datepicker)
    • Wo (Adresse) (Map mit Böbbel)
    • Maximale Teilnehmeranzahl festlegen
    • Beschreibung des Events

  • Eingeloggte User können ein Event bearbeiten

  • Eingeloggte User können ihr Profil bearbeiten

    • Bild
    • Name
    • Email
    • Benachrichtigungen einstellen
